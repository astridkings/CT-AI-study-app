# Companheiro de Estudos CT-AI

Aplicativo web para estudo da certificação ISTQB Certified Tester AI Testing (CT-AI).

## 📚 Sobre

Este app oferece:
- Plano de estudos estruturado (11 capítulos)
- Resumos completos de cada capítulo
- Quizzes interativos (10 questões por capítulo)
- Provas simuladas (3 exames completos)

Baseado no programa oficial ISTQB CT-AI v1.0.

## 🚀 Como usar

### Opção 1: Executar localmente
1. Baixe/clone este repositório
2. Abra `index 1.html` no navegador
3. Navegue pelas seções: Início, Plano de Estudos, Resumos, Quizzes, Provas Simuladas

### Opção 2: Acessar online (GitHub Pages)
Visite: [https://seu-usuario.github.io/ct-ai-study](https://seu-usuario.github.io/ct-ai-study)

## 📁 Estrutura do projeto
- `index 1.html` - Página principal
